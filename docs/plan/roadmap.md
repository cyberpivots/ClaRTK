# ClaRTK Roadmap

## Phase 0

- Scaffold repo manifests, agent policy, ADRs, task tracking, and vendor/legal notes.

## Phase 1

- Implement contracts, GNSS protocol crates, device adapters, solver boundaries, fixtures, and RTKLIB patch flow.

## Phase 2

- Implement runtime services, runtime database schema, and browser dashboard.

## Phase 3

- Implement unified React Native app for iOS, Android, and Windows on the common supported line.

## Phase 4

- Implement development/agent-memory services, embeddings/evals, and ML/mapping pipelines.
